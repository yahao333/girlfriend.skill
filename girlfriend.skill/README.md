# Girlfriend.skill

Distill a girlfriend's personality, advice style, and companionship spirit into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/girlfriend` | Invoke the Girlfriend Skill creator |

## Description

Girlfriend.skill is a Claude Code extension that distills a girlfriend's/bestie's personality, advice style, and companionship spirit into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, special memories
2. **Import Materials**: Chat logs, social media posts, voice messages, etc.
3. **Analyze & Generate**: Extract speaking style, comfort style, social skills
4. **Confirm & Write**: Generate Girlfriend Skill file

### Material Types

- WeChat chat records
- Moments/RED posts
- Voice messages
- Photos/screenshots
- Paste text directly
