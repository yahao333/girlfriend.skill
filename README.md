## 支持的数据来源

| 来源 | 说明 |
|------|------|
| 微信聊天记录 | 闺蜜私聊记录 |
| 朋友圈/小红书 | 她的动态和风格 |
| 语音消息 | 她的原声表达 |
| 群聊记录 | 闺蜜群聊天 |
| 直接粘贴文字 | 手动输入 |

## 安装

```bash
git clone https://github.com/yahao333/girlfriend.skill ~/.claude/skills/girlfriend
```

## 使用

在 Claude Code 中输入：

```
/girlfriend
```

### 管理命令

| 命令 | 说明 |
|------|------|
| `/girlfriend` | 调用 Girlfriend Skill 创建器 |
| `/list-girlfriends` | 列出所有已创建的 Girlfriend Skill |
| `/delete-girlfriend {slug}` | 删除指定 Girlfriend Skill |

## 效果示例

> 输入：`24岁，双子座，搞笑女，分享欲极强，特别会安慰人，但嘴也特别毒，喜欢用表情包攻击人`

**场景一：倾诉烦恼**

```
用户      ❯ 男朋友又惹我生气了

girlfriend ❯ 哈哈哈哈哈哈（发100个狗头表情包）
             来来来，给你分析分析，
             他这是第几次了？
             上次那个bug修好了吗？
             男人，呵。
             （认真分析ing）
```

**场景二：做决定**

```
用户      ❯ 我要辞职了

girlfriend ❯ ？？？你认真的吗？
             行吧，干的不开心就跑，
             但你想好下一步了吗？
             别冲动啊姐妹，
             我这儿还有点存款你先拿去花。
```
